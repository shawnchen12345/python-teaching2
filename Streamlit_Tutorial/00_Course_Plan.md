# Streamlit 8小时极速入门教程

欢迎来到 Python Web 开发的世界！不需要复杂的前端知识 (HTML/CSS/JS)，只要会写 Python，你就能做出漂亮的网页应用。

**主要内容**:
1.  **极速入门**: 30分钟写出你的第一个网页。
2.  **常用组件**: 按钮、输入框、滑块等。
3.  **页面布局**: 侧边栏、多列布局，让网页更美观。
4.  **状态管理**: 让网页拥有“记忆”，不再是点一下就重置。
5.  **数据展示**: 图表、表格、图片。
6.  **最终项目**: 动手做一个可以在手机上使用的“个人任务管理器”。

---

## 学习路径

### 第一阶段：基础 (01_hello.py)
-   安装 Streamlit: `pip install streamlit`
-   文本显示: Title, Header, Write
-   基础交互: Button, TextInput, Slider

### 第二阶段：布局与状态 (02_layout_and_state.py)
-   Sidebar (侧边栏)
-   Columns (多列)
-   **核心难点**: Session State (会话状态)

### 第三阶段：数据可视化 (03_data_display.py)
-   展示表格 (DataFrame)
-   绘制图表 (Line Chart, Bar Chart)
-   展示图片

### 第四阶段：实战项目 - 任务管理器 (04_project_template.py)
-   综合运用所有知识，完成一个 To-Do List 应用。

---

## 如何运行代码？

在终端 (Terminal) 中输入：

```bash
streamlit run <文件名.py>
```

例如：
```bash
streamlit run 01_hello.py
```

网页会自动在浏览器中打开。
