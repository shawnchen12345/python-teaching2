# 🐍 Python 面向对象编程 (OOP) 完整课程大纲

**总课时**: 约 8 小时 (4 次课，每次 2 小时)  
**适用对象**: 已学完 Python 基础（变量、数据类型、判断/循环、函数、列表/字典）的学生  
**教学目标**: 系统掌握面向对象编程思想，能用 OOP 思维设计和架构中型项目  
**教学方式**: 讲解 + 现场编码演示 + 课堂练习 + 课后作业

## 📝 教学建议

### 课前准备
1. 确保学生电脑已安装 Python 3.8+ 和 VS Code
2. 提前打开讲义和代码文件
3. 准备一个空白 Python 文件用于现场编码演示

### 课堂技巧
1. **先运行后讲解**: 先展示代码运行效果，引发好奇心，再解释原理
2. **故意犯错**: 演示常见错误（忘写 self、__init__ 拼错等），让学生印象深刻
3. **类比第一**: 每个概念都先用生活类比引入，再讲编程
4. **循序渐进**: 从最简单的类开始，逐步增加属性和方法
5. **鼓励提问**: "有没有不清楚的地方？""谁能猜一下这段代码会输出什么？"
6. **节奏控制**: 核心知识点讲完后休息 2-3 分钟

### 评估方式
- **课堂观察**: 学生是否能跟上现场编码
- **练习完成度**: 基础练习（1-3题）必须完成
- **挑战题**: 学有余力的同学完成，可加分
- **课后大项目**: 第4课结束后的综合项目，作为整体评估
